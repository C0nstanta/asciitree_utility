# asciitree_utility
##c++ asciitree utility
<p>The utility transforms a string like this</p>

 <strong>"(asciitree (sometimes you) (just (want to draw)) trees (in (your terminal)))"</strong>


In a tree of this format

<p>asciitree<br>
<p>+--sometimes</p><br>
<p>|   +--you</p><br>
<p>+--just</p><br>
<p>|   +--want</p><br>
<p>|      +--to</p><br>
<p>|      +--draw</p><br>
<p>+--trees</p><br>
<p>+--in</p><br>
<p>|   +--your</p><br>
<p>|      +--terminal</p><br>
